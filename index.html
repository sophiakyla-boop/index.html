What I Learned About HTML & CSS

My Introduction to Computing subject included an introduction to Computers, the basic fundamentals such as CPU, memory as well as the storage devices and how they work together to accomplish their tasks. I also learned how the Operating System manages the resources and acts as the mediator between hardware and software with respect to accessing resources.

Then came algorithms and problem-solving, which helped me learn how to step-by-step break down a task. As I continued, I gained knowledge about the history of computers, various types of software, and the significance of networking through the Internet.

Finally, arrived at the lesson of HTML/CSS that deals with methods for creating webpage structures using HTML and formatting as well as styling using CSS to make it presentable. This experience has surely helped me gain a broad perspective of the field of computing and more importantly, has helped me to understand the basics of web development

Overcoming Challenges

I enjoyed HTML and CSS from Introduction to Computing. What I liked about HTML was that I could build the structure of a web page using elements like headings, paragraphs, and links, and then I could style the parts created using those with CSS for wonderful and more pleasing designs. It's great fun to see how small changes in the code can make a dramatic change in the front end, and I enjoyed playing with these layouts and color schemes.

The most difficult lesson learned was also in HTML and CSS but this time, due to a different cause. Sometimes it got tough to understand how exactly to apply CSS properly to style elements including how to handle positioning, margins, and responsiveness for different view screens. Therefore, this process required so much trial-and-error to satisfy the outcome, particularly to ensure that pages look beautiful both on desktop and mobile view screens

Having gone through this, though, was what made the complexity and flexibility of web design so appealing.
